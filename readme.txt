== Description ==

WordHub helps you publish various projects from your GitHub account. Just add your GitHub username and API token to the settings panel, and you're good to go.

== Installation ==

The best way to install WordHub is via the Add New link under Dashboard > Plugins.

To install WordHub manually, follow these steps.

1. Upload the `wordhub` directory to `/wp-content/plugins/`
2. Activate WordHub through the WordPress 'Plugins' menu
3. Visit Dashboard > WordHub to set your username and API token, and begin using WordHub on your WordPress site.
